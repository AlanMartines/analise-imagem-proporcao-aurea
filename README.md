<h2 id="sobre">📜 Sobre</h2>
<p>
    Este projeto foi desenvolvido para analisar imagens usando a proporção áurea e a regra dos terços. Com ele, é possível visualizar as linhas guias dessas proporções diretamente na imagem, auxiliando em análises de composição e design.
</p>

<hr>

<h2 id="pre-requisitos">🛠 Pré-requisitos</h2>
<p>
    Antes de começar, você vai precisar ter instalado em sua máquina:
    <a href="https://www.python.org/downloads/">Python</a> e as bibliotecas <a href="https://pypi.org/project/opencv-python/">OpenCV</a> e <a href="https://pypi.org/project/Pillow/">Pillow</a>.
</p>

<hr>

<h2 id="como-usar">🚀 Como usar</h2>
<ol>
    <li>Clone o repositório</li>
    <pre><code>git clone https://github.com/SeuUsuario/analise-imagem-proporcao-aurea.git AnaliseImagemProporcaoAurea</code></pre>
    <li>Navegue até o diretório do projeto</li>
    <pre><code>cd AnaliseImagemProporcaoAurea</code></pre>
    <li>Execute o script</li>
    <pre><code>python analise_imagem.py</code></pre>
    <li>Pronto, imagem analisada</li>
</ol>
